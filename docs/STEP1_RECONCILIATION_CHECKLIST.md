## B. Звірка цифр із 1С (головне) — по одному періоду

### B1. Борг клієнтів (найважливіше)

- **У нас:** Звіти → Прострочені борги (загальна сума) + картка 2-3 конкретних клієнтів (поле «Борг»).
- **У 1С:** «Ведомость взаиморасчётов з контрагентами» / борг по тих самих клієнтах на ту саму дату.
- **Звіряємо:** (а) борг 2-3 іменних клієнтів; (б) загальна сума дебіторки.
- **Стан:** ✅ збігається. Звірено Гуцу Тетяна (період 01.04–31.05.2026): наш регістр
  Борг (рухи) = **+95.38 €**, 1С «Ведомость по взаиморасчётам» кон. остаток (EUR) = **95.38**.
  Окремі рухи теж один-в-один (170.13/9.82/160.31/105.00/9.62). DEBT_SIGN=1 підтверджено.

### B2. Продажі — виручка і кг за період

- **У нас:** Звіти → Підсумок продажів (за обраний місяць): загальна сума + загальна вага (кг) + топ-агент.
- **У 1С:** звіт «Продажи» (Обороти) за той самий місяць.
- **Звіряємо:** виручку за місяць; вагу (кг) за місяць; суму по 1-2 торгових агентах.
- **Стан:** ⚙️ Звіт «Підсумок продажів» перероблено на **гнучкий** (дерево, як 1С «Продажи»,
  2026-06-23): показники (К-сть/Вага/Виручка/Виручка без знижок/Знижка), багаторівневі
  групування→дерево (Товар/Клієнт/Область/Місто/Агент/КатТТ/ТипЦін/Замовлення/ДокПродажу/Рік/Місяць),
  фільтри, загальні підсумки, XLSX/CSV. **Виправлено:** ліміт 5000 (тепер весь період), товари
  назвами (id|code), агент = Контрагент.ТорговийАгент (через клієнта). Дубль «Продажі по клієнтах»
  прибрано. **Звірка сум із 1С — TODO user** (обрати місяць, звірити виручку €/кг/агентів).

### B3. Маржа / валовий прибуток

- **У нас:** Звіти → Маржа (за період): Виручка − Собівартість = Валовий прибуток.
- **У 1С:** «Валовая прибыль» за той самий період.
- **Звіряємо:** валовий прибуток за місяць; маржа по 1-2 топ-товарах.
- **Стан:** ⚙️ Звіт «Маржа/Валовий прибуток» перероблено на **гнучкий** (дерево, той самий движок,
  що й продажі, 2026-06-23): показники Виручка €/Собівартість €/Валовий прибуток €/**Маржа %**
  (Маржа% — похідна, рахується на кожному вузлі, не сумується); багаторівневі групування→дерево
  (Товар/Категорія/Клієнт/Область/Місто/Агент/Рік/Місяць); фільтри; XLSX/CSV. Джерело незмінне
  (виручка posted-реалізацій + CostMovement). **Звірка з 1С «Валовая прибыль» — TODO user.**

### B4. Рух коштів (каса/банк, ДДС)

- **У нас:** Звіти → Рух коштів (за період): прихід / розхід / сальдо по статтях.
- **У 1С:** «Движения денежных средств» за той самий період.
- **Звіряємо:** загальний прихід і розхід за місяць; 1-2 найбільші статті.
- **Стан:** ⚙️ Звіт перероблено на **гнучкий** (дерево, показники, фільтри) +
  **мультивалютний** (2026-07-02): окремі колонки грн/євро/долар (Сумма за валютою
  рахунку) + «упр. €» (СуммаУпр). Ієрархія папок статей ДДС (Категорія/Підкатегорія).
  Потребує реімпорту `--entity cashflow-reg` (валюта) + `--entity dictionaries`
