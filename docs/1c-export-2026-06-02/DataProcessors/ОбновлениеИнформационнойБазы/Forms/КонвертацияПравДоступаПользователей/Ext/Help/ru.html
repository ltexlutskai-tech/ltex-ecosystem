Обработка «Конвертация прав доступа пользователей»

Обработка используется в контексте обновления на ред. 2.3

Обработка производит конвертацию прав доступа пользователей к объектам для учета изменений в системе ограничений прав доступа, которые произведены в редакции 2.3 по сравнению с редакцией 2.2
