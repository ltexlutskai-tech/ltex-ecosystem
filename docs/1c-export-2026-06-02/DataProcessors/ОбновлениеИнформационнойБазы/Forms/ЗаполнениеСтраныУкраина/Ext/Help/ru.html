Обработка «Заполнение страны Украина»

Обработка используется в контексте обновления на ред. 2.3

В справочник «Классификатор стран мира» добавлен предопределенный элемент «Украина». Данная обработка осуществляет обновление данных о стране в объектах информационной базы. Обработка может выполняться в несколько этапов, либо все этапы могут быть выполнены в один прием.

Обновить адреса контактной информации

На данном этапе происходит обновление записей регистра сведений «Контактная информация», которые имеют тип «Адрес». Для всех Адресов не из Украины в «представление» добавляется страна
