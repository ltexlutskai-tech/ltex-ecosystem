Отчет "Отчет о суммах входящих налоговых документов"

Отчет предназначен для анализа полноты и правильности регистрации входящих налоговых документов (документы "Запись книги приобретения").

 

Для построения отчета по операциям возврата необходимо выбрать соответствующее значение переключателя («Возврат») на форме отчета. Для анализа операций отгрузки товаров необходимо выбрать значение переключателя «Расчеты».

 

В отчете отдельно выводится информация, связанная с взаиморасчетами за товары (услуги и т. п) и за возвратную тару.

  

В названиях показателей применено условное сокращение «НД» = «Налоговые документы»

 

Показатели отчета:

       Не выписанные/излишне выписанные НД на начало – сумма взаиморасчетов  на начало периода, на какую не были зарегистрированы налоговые документы (если значение показателя положительно), либо налоговые документы зарегистрированы излишне (если значение показателя отрицательно).

 

       Сумма не закрытых отгрузок (оплат) на начало (сумма взаиморасчетов),

       Сумма не закрытых отгрузок на начало (сумма входящего НДС)  – эта группа показателей показывает сумму поставок (или авансовых оплат) по которым еще не произошло оплаты (отгрузки соответственно) на начало периода. Значение данных показателей может быть некорректно в случае, когда показатель "Не выписанные/излишне выписанные НД на начало" не равен нулю.

 

       Сумма взаиморасчетов, требующая подтверждения НД за период – показатель расчитывается с учетом момента определения базы НДС, заданного в Договоре контрагента (в реквизитах «схема налогового учета» и «схема налогового учета по таре»). Данный показатель заполняется только в строке отчета, соответствующей оплате.

 

       Сумма отгрузок, требующая подтверждения НД за период – данный показатель заполняется только в строках отчета, соответствующих отгрузкам.

      

       Выписано НД за период (сумма взаиморасчетов),

       Выписано НД за период (сумма входящего НДС) – показатели заполняются по данным зарегистрированых в течение периода налоговых документов. Информация в отчет выводится  отдельными строками в разрезе ставок НДС. Информация выводится в строках отчета, соответствующих отгрузкам.

 

       Сумма взаиморасчетов, требующая подтверждания НД на конец – показатель расчитывается с учетом момента определения базы НДС, заданного в Договоре контрагента (в реквизитах «схема налогового учета» и «схема налогового учета по таре»). Данный показатель заполняется только в строке отчета, соответствующей оплате.

 

       Сумма не закрытых отгрузок (оплат) на конец (сумма взаиморасчетов),

       Сумма не закрытых отгрузок на конец (сумма входящего НДС)  – эта группа показателей показывает сумму поставок (или авансовых оплат) по которым еще не произошло оплаты (отгрузки соответственно) на конец периода. Значение данных показателей может быть некорректно в случае, когда показатель "Не выписанные/излишне выписанные НД на конец" не равен нулю.
