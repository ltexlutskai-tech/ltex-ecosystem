Отчет "Проверка суммы входящего НДС"

Отчет предназначен для анализа соответствия даты налоговых документов "Запись книги приобретения"  дате возникновения налогового кредита.

 

В отчете отдельно выводится информация, связанная с взаиморасчетами за товары (услуги и т. п) и за возвратную тару.

 

Период формирования отчета устанавливается кратным месяцу.

 

Для построения отчета по операциям возврата необходимо выбрать соответствующее значение переключателя («возврат») на форме отчета. Для анализа операций отгрузки товаров необходимо выбрать значение переключателя «отгрузка».

 

Показатели отчета (в грн., суммы включают НДС):

 

      Сумма взаиморасчетов, требующая подтверждения НД за период – показатель рассчитывается с учетом момента определения базы НДС, заданного в Договоре контрагента (в реквизитах «схема налогового учета» и «схема налогового учета по таре»).  

      Получено НД за период (сумма взаиморасчетов) – показатель заполняется по данным зарегистрированных в течение периода налоговых документов. 

 

 

Замечания:

1. Для анализа правильности регистрации налоговых документов (ставки НДС, счета учета НДС и проч.) необходимо воспользоваться отчетом  Отчет о суммах входящих налоговых документов.
2. В отчет выводится информация обо всех документах, участвующих в налоговом учете НДС.
