### Push protocol –і–ї—П orchestrator-—Б–µ—Б—Ц—Ч (cloud Claude / sandbox)

**–Т–Р–Ц–Ы–Ш–Т–Ю:** –Я–Њ—В–Њ—З–љ–∞ (cloud Claude / Sandbox) —Б–µ—Б—Ц—П = orchestrator. –Э–µ–Ј–∞–ї–µ–ґ–љ–Њ –≤—Ц–і –љ–∞–Ј–≤–Є –њ—А–Є—Б–≤–Њ—Ф–љ–Њ—Ч branch —Г system-prompt (–љ–∞–њ—А–Є–Ї–ї–∞–і `claude/ltex-orchestration-XXXX`) вАФ –Э–Х —А–Њ–Ј—А–Њ–±–ї—П–є —Г –љ—Ц–є. –†–Њ–±–Њ—В–∞ orchestrator-–∞: merge worker branches —Г `main` + –њ–Њ–Љ—Ц—З–љ—Ц —Б–Ї—А–Є–њ—В–Є inline, –њ–Њ—В—Ц–Љ **push –њ—А—П–Љ–Њ –≤ main**.

Sandbox proxy –њ–µ—А—Ц–Њ–і–Є—З–љ–Њ 403-–љ–Є—В—М `git push origin main` (recurring environmental bug, —Б–њ–Њ—Б—В–µ—А—Ц–≥–∞—Ф—В—М—Б—П –Ј S82). **–Я—А–Њ—В–Њ–Ї–Њ–ї:**

1. **Try direct main push first:**
   ```bash
   git checkout main
   git merge --no-ff origin/<worker-branch> -m "Merge ..."
   git push origin main
   ```
2. **–ѓ–Ї—Й–Њ –Њ—В—А–Є–Љ–∞–≤ HTTP 403 / "RPC failed":** fallback –љ–∞ side-branch –і–ї—П local fast-forward:

   ```bash
   git push origin main:claude/sNN-merge   # NN = session number
   ```

   –Ґ–Њ–і—Ц user –ї–Њ–Ї–∞–ї—М–љ–Њ –љ–∞ Windows:

   ```powershell
   git fetch origin
   git checkout main
   git merge --ff-only origin/claude/sNN-merge
   git push origin main
   git push origin --delete claude/sNN-merge
   ```

3. **–Э–Х –њ—Г—И–Є—В–Є —А–Њ–±–Њ—В—Г –љ–∞ feature-branch —П–Ї—Й–Њ —Ж–µ orchestrator session.** Worker sessions вАФ —Ц–љ–∞–Ї—И–µ; –≤–Њ–љ–Є –Љ–∞—О—В—М —Б–≤–Њ—О dedicated feature branch.

4. **Sandbox stop-hook** —Б–Ї–∞—А–ґ–Є—В—М—Б—П –љ–∞ unpushed main commits. –ѓ–Ї—Й–Њ 403 –Ј–∞–±–ї–Њ–Ї—Г–≤–∞–≤ push вАФ switch —Б–µ—Б—Ц—Ч –љ–∞–Ј–∞–і –љ–∞ —Ц–љ—И—Г –≥—Ц–ї–Ї—Г (`git checkout claude/<something>`) —Й–Њ–± hook –Ј–∞–Љ–Њ–≤–Ї; main commits –ґ–Є–≤—Г—В—М –љ–∞ side-branch.
